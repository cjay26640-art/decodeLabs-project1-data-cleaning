# Data Cleaning & Preparation Project

This project was completed as part of the Data Analytics Industrial Training Program by DecodeLabs.

## Objective
The goal of this project was to clean and prepare raw data for analysis.

## Tasks Performed
- Checked missing/null values
- Verified duplicate records
- Validated Order IDs
- Confirmed date formatting consistency
- Reviewed numeric and text columns
- Prepared clean and analysis-ready dataset

## Tools Used
- Microsoft Excel

## Final Result
The dataset was successfully cleaned and prepared for further analysis.
